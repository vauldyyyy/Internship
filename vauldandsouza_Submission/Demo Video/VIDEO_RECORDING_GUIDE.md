## RECORDING INSTRUCTIONS

### Using OBS Studio (Recommended)

1. **Open OBS**
2. **Add Source:**
   - Click `+` under Sources
   - Select **Display Capture** or **Window Capture**
   - Choose browser window showing Streamlit dashboard

3. **Audio Setup:**
   - Devices → Microphone: Select your microphone
   - Test audio level (should be green/yellow, not red)

4. **Quality Settings:**
   - Settings → Video
   - Base Resolution: 1920x1080
   - Output Resolution: 1920x1080
   - FPS: 30
   
   - Settings → Output
   - Bitrate: 8000-10000 Kbps
   - Audio Bitrate: 192 Kbps

5. **Recording:**
   - Click **Start Recording**
   - Follow script (8-10 minutes)
   - Click **Stop Recording**
   - File saved to: Documents/Videos or configured folder

### Post-Production:

**If editing:**
1. Open video in DaVinci Resolve (free) or Camtasia
2. Remove long pauses/mistakes
3. Add title card (2 seconds): "Nifty 100 Financial Intelligence"
4. Add ending card (2 seconds): "Thank you for watching!"
5. Add optional background music (low volume, royalty-free from YouTube Audio Library)
6. Export as MP4 (H.264, 1080p)

**Final File:**
- Size: 150-300 MB (8-min video)
- Format: MP4
- Location: `Demo Video/dashboard_demo.mp4`
